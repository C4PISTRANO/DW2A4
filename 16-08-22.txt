Document Object Model - DOM:
representação da estrutura dentro da HTML document

*para acessar o document no java, utilizaremos "window.document"

para recuperar elementos no HTML:
id
classe
tag
atribute name
seletor CSS

Em CSS:
# = buscando por id
. = buscando por classe

Eventos do Javascript:
A interatividade entre o usuário e uma página web depende fortemente
dos eventos (interação do usuário com a página)

Normalmente esses eventos são associados a uma função

*como boas práticas a ação deve ser colocada no Javascript
